Research experience

roles

Ph.D. Student, Knowledge-based Data Discovery Lab, Inha University

Sep 2023 – Present · Incheon

Designed a real-time, high-performance, low-cost Transformer-based architecture for IoT devices. Explored explanations in recommender systems with large language models and analyzed their impact on recommendation performance. Developed an explainable Korean text analyzer using deep learning. Enhanced knowledge graphs for recommender systems to reflect unstructured data. Proposed a personalized generation approach using a diffusion model to reduce user interventions. Integrated continual learning considering dynamic and static preferences in recommender systems.

Built pipelines for data collection, analysis, modeling, and experimentation for IoT products. Conducted comparative studies on Korean morpheme analyzers to improve text cohesion.

Undergraduate Research Student, Knowledge-based Data Discovery Lab, Inha University
